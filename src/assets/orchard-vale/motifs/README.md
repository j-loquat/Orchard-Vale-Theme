# Orchard Vale App-Specific Motif Packs

Category 11 asset pack for Orchard Vale HTML front-ends.

These SVG motifs provide small domain-specific visual vocabulary for cards, sidebar sections, tabs, empty-state accents, and dashboard modules. They are decorative support assets, not primary controls.

## Usage Guidance

- Use these at `32px`, `48px`, `64px`, or `96px` depending on density.
- Keep labels and critical meaning in HTML text; motifs should reinforce domain context.
- Prefer inline SVG if you need color-state changes, or `img` tags for simple decoration.
- Use no more than one or two motif families per screen unless the product flow truly spans multiple domains.
