## Usage

Use crests for app identity, workspace identity, team identity, section marks, and product-area badges. Use seals for status marks and review/approval states. Use the laurel, crown, and blank shield frame as composable decorative pieces behind real HTML labels or icons.

Avoid using these as full interactive controls by themselves. They should decorate or identify real HTML elements.
