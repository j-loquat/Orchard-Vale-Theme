# Orchard Vale Functional Icons

Category 5 asset pack for Orchard Vale HTML front-ends.

These are crisp, scalable SVG icons designed for real UI controls. They use a consistent dark ink outline, parchment/brass/green fill language, and restrained semantic colors for status/action meaning.

## Usage

- Use the SVGs directly as `img` sources, CSS masks, or inline SVG when color/state control is needed.
- Target display sizes: `24px`, `32px`, and `48px`.
- Pair icons with HTML text labels for dense controls unless the action is globally familiar.
- Keep status icons color-coded; avoid recoloring warning/error/success states unless theme tokens require it.
