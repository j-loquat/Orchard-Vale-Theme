## Usage Guidance

- Use avatar portraits in rosters, comments, owner chips, assistant panels, and guild/team selectors.
- Use full-body helpers for side panels, empty states, onboarding, celebratory states, and app-specific guidance.
- Keep HTML text separate from the images; these assets intentionally avoid baked-in labels.
- Place characters on parchment or light patterned surfaces when possible; avoid dark busy backdrops behind detailed poses.
- Use one character at a time in dense product screens so the theme stays mature and readable.
