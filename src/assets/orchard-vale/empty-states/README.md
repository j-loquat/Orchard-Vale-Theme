# Orchard Vale Empty States And Spot Illustrations

Category 8 asset pack for Orchard Vale HTML front-ends.

This pack contains transparent PNG illustrations for empty queues, no-results screens, completed workflows, maintenance states, notification panels, and small decorative moments. Text should remain real HTML layered beside or above these assets.

## Usage Guidance

- Use `800x600` empty states in wide panels, dashboard blanks, no-results screens, and completion states.
- Use the left-side transparent space for real HTML headings, explanatory copy, and calls to action.
- Use `512x512` spot illustrations inside cards, sidebars, help panels, and compact onboarding moments.
- Keep the image decorative and the UI semantic; do not rely on the illustration to convey critical state alone.
- Avoid placing these over busy dark backgrounds; parchment panels or soft washes preserve detail.
