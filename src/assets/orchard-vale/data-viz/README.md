# Orchard Vale Data Visualization Ornament

Category 6 asset pack for Orchard Vale HTML front-ends.

These assets are SVG ornaments for real charts, tables, timelines, meters, and tooltips. They should frame or support data visualization without replacing semantic HTML, live SVG data marks, accessible labels, or chart-library output.

## Usage Guidance

- Use chart frames and plot backgrounds outside the plotted data area.
- Use gridlines, ticks, and label plaques sparingly so the data stays readable.
- Use meter tracks, fills, and endcaps as layered CSS backgrounds or inline SVG pieces.
- Use timeline plaques behind real HTML labels, not with text baked into the asset.
- Use tooltip shells as CSS background images or inline SVG containers with real HTML content on top.
- Use category symbols for legends and compact chart annotations.
