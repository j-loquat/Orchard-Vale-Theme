# Orchard Vale Scene Backgrounds

Category 9 asset pack for Orchard Vale HTML front-ends.

This pack contains wide environmental backgrounds for dashboards, empty states, onboarding, login pages, and map-like work areas. They are designed to support real HTML panels over the image, not replace the interface.

## Usage Guidance

- Use `wide-16x9` assets for dashboard hero areas, empty-state panels, login/onboarding screens, and large app shell backgrounds.
- Use `ultrawide-21x9` assets for full-bleed desktop bands, wide monitor shells, or horizontally constrained hero panels.
- Add a parchment, green, or subtle dark overlay behind text-heavy panels if the background competes with content.
- Keep actual labels, controls, charts, and navigation in HTML/CSS; the images intentionally avoid baked-in copy.
- Prefer `object-fit: cover` with stable focal positioning per scene rather than stretching the images.
